# 커밋 메시지 제안

## 초기 커밋 (전체 프로젝트)

### 제목
```
feat: 자격증 공부 기록 캘린더 애플리케이션 구현
```

### Description
```
CertFlow - GitHub 잔디 그래프 스타일의 자격증 공부 기록 관리 시스템

주요 기능:
- GitHub 스타일 캘린더로 공부 기록 시각화
- 회원별 공부 기록 관리 및 필터링
- 전체 월별 공부 기록 표시
- 반응형 디자인 (모바일/데스크톱)
- 모바일 클릭 인터랙션 지원

기술 스택:
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS

SEO 및 접근성:
- 메타데이터 최적화
- Open Graph 태그
- 구조화된 데이터 (JSON-LD)
- ARIA 레이블 및 시맨틱 HTML
- robots.txt 및 sitemap.xml 자동 생성
- PWA 매니페스트

날짜 범위: 2026-2027년
```
